General Information

  • A personalized calendar at the topleft.
  • A news feed on the top right.
  • I created a notification hub underneath the news that includes text messages, social messages and information, and just basic notification updates.
  • I included local weather information right underneath the calendar because i believed the two fit near each other.

  • I added a Date and Time indicator at the top middle of the screen so the user can easily spot that information.
  • Lastly I added two buttons to the bottem middle of the screen for the user, one button is for music because who doesnt want music in their morning routines and also a power button to turn off the additional features and just use a regular mirror.

Important things to note on how my design works and how to run the code!:

  • In order to run the code, you must complete the following steps.
  • Step 1: You must import all the files from the repository involving CSV files, Helvetica-Bold.ttf, as well as the Sunday-morning-mp3.
  • Step 2: You must make sure the processing ide has the correct sound library installed, to do this go to tools -> manage tools -> install sound library, in this case you must install the Sound Library - author is the The Processing Foundation.
  • Step 3: You must make sure the processing ide has the correct video library installed, to do this go to tools -> manage tools -> install Video library, in this case you must install the video library for processing 4 Library - author is the The Processing Foundation.

Implementations Explanations

I created csv files using Microsoft Excel for my personalized calendar, news feed, local weather and notifications for the general information provided on the mirror.

I also created another csv file for the health information on the magic mirror for the case of weight and how it compares to the previous weeks data.

I was able to implement these to the processing sketch by creating table objects in processing then loading the specific CSV file to each table variable and outputing them to the final sketch.

  • Little side note, the Microsoft Excel files were saved as CSV UTF-8 (Comma Delimited)(*.csv) files.

I added some personal utilities to my designs as well

  • At the top of the screen, I created a date for the user that updates and provides the Month, Day, and Year using the built in processing functions .
  • At the top of the screen next to the date, I created the time for the user as well that updates and provides the exact time down to the seconds for the users who could maybe be running late, this was done using the built in processing functions.
  • I also imported the video library to help give the mirror a more realistic feel to it.
  • Utilizing my camera which was a Logitech 1080p C920s Pro HD Webcam, I was able to create a mirror like reflection in my design, to show show how a user can be reflected while all the information is provided on the screen that the user would want.

Out of the options available, I choose to use my imagination on my design to further expand additional elements I would want my mirror to have

I included a music button where it would play music for the user in the morning or at night when doing basic routines, it gives it a sense of realism.

In addtion to the music button I also created a power button at the bottem middle of the mirror to clean the mirror screen and to just allow the user to be able to look at themselves without all the additional information.

This power button clears the screen only allowing the camera reflection, so users can just use the mirror without any additional information at there disposal giving the mirror more simplicity.
